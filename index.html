What does Skeleton do?

Skeleton is not a framework. It is a CSS boilerplate that allows you to quickly set up beautiful CSS projects. It gives you a few useful and powerful CSS Classes that you can use for your website.

Skeleton Versions

After you've downloaded the Skeleton zip file, unpack it in your preffered location.

In the unzipped folder, you'll find the index.html file. This is the file you'll have to edit to make your website. Ofcourse you can remove this one, and make your own .html or .php file.
